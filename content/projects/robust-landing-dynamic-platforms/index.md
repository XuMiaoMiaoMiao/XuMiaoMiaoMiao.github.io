Developing a robocentric-frame differential model for UAV landing on dynamic platforms using relative visual information between robots.
The project investigates L1-adaptive MPC to estimate unobservable platform variables such as acceleration and angular acceleration, aiming to improve landing robustness under target motion uncertainty.
